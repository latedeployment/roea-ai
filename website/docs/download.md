## System Requirements

| Platform | Minimum Version |
|----------|-----------------|
| **macOS** | 11.0 (Big Sur) or later |
| **Windows** | Windows 10 (1903) or later |
| **Linux** | Kernel 4.18+, glibc 2.17+ |

For the best experience on Linux, use kernel 5.8+ with eBPF support.

See [full system requirements](/guide/requirements) for details.

## Installation

### macOS

1. Download the `.dmg` file for your chip (Apple Silicon or Intel)
2. Open the DMG and drag roea-ai to Applications
3. Launch from Applications or Spotlight

::: tip First Launch
On first launch, macOS may ask for permissions. Grant "Full Disk Access" for file monitoring.
:::

### Windows

1. Download the `.msi` installer
2. Run the installer and follow the prompts
3. Launch from Start Menu

::: tip Windows Defender
Windows may show a SmartScreen warning on first run. Click "More info" ‚Üí "Run anyway".
:::

### Linux

#### Debian/Ubuntu

```bash
# Download and install
curl -LO https://github.com/your-org/roea-ai/releases/latest/download/roea-ai_amd64.deb
sudo dpkg -i roea-ai_amd64.deb
sudo apt-get install -f  # Fix dependencies if needed

# Start the daemon
roea-agent
```

#### Other Distributions

```bash
# Download standalone binary
curl -LO https://github.com/your-org/roea-ai/releases/latest/download/roea-agent-linux-x64
chmod +x roea-agent-linux-x64
sudo mv roea-agent-linux-x64 /usr/local/bin/roea-agent

# Start the daemon
roea-agent
```

#### Enable eBPF (Optional, Recommended)

For best performance on Linux:

```bash
# Grant BPF capability
sudo setcap cap_bpf+ep /usr/local/bin/roea-agent

# Or run with sudo
sudo roea-agent
```

## Verify Downloads

All releases include SHA256 checksums. Verify your download:

```bash
# Download checksums
curl -LO https://github.com/your-org/roea-ai/releases/latest/download/checksums.txt

# Verify (Linux)
sha256sum -c checksums.txt --ignore-missing

# Verify (macOS)
shasum -a 256 -c checksums.txt --ignore-missing
```
