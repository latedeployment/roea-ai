# roea-ai Project Status - 2026-01-13

## Overview

**roea-ai** is an EDR-like observability tool for AI coding agents (Claude Code, Cursor, Copilot, Aider, Windsurf, etc.). It monitors processes, network connections, and file access to give developers visibility into what AI agents are doing on their systems.

## Local AI Model Support (Ollama)

You can use local LLMs for development. Compatible models:

| Model | Size | Use Case |
|-------|------|----------|
| `qwen2.5-coder:7b` | ~4GB | Strong coding, recommended |
| `codellama:13b` | ~7GB | Good code generation |
| `deepseek-coder:6.7b` | ~4GB | Fast completion |
| `mistral:7b` | ~4GB | General purpose |
| `llama3:8b` | ~4.7GB | Balanced |

### Setup

```bash
# Install Ollama
curl -fsSL https://ollama.com/install.sh | sh

# Pull a model
ollama pull qwen2.5-coder:7b

# Use with aider
aider --model ollama/qwen2.5-coder:7b

# Environment variable
export OLLAMA_API_BASE=http://localhost:11434
```
